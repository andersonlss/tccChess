EXAMPLE1
========
This is a sample program to help C/C++ programmers get
started using the Robix Nexway library (NexwayLib).

Contents
--------
/src           - example1 source files
/VC6           - project files for building the example with 
                 Microsoft Visual C++ 6
/VC.NET2003    - project files for building the example with 
                 Microsoft Visual C++ .NET 2003
